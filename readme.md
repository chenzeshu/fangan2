## 项目介绍

本项目旨在：<br>
1、为企业提供一个多人维护的统一产品数据库；<br>
2、在线以“从产品库中选取产品”的形式编辑标书，减少手工劳作；<br>
3、方案（标书）撰写人员可以获得一个专属的方案空间，用于存放历次编写的标书；<br>
4、实现设备清单拖拽排序；<br>
5、提供excel及word导出功能；
